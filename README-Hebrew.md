# מערכת ניהול חברת תחבורה

## סקירה כללית
מערכת ניהול חברת תחבורה היא פלטפורמה מקיפה המיועדת לניהול שירותי תחבורה בין חברות ונהגים. המערכת מאפשרת לחברות תחבורה ליצור בקשות נסיעה, לנהל נהגים ולעקוב אחר סטטוס הנסיעות, בעוד שנהגים יכולים לצפות בנסיעות זמינות, לקבל או לדחות בקשות נסיעה ולעדכן את זמינותם.

## תכונות מרכזיות
- **ניהול משתמשים**: רישום, אימות והרשאות למשתמשים שונים (מנהל, חברה, נהג)
- **ניהול פרופיל**: עדכון ושמירת פרטי פרופיל לחברות ונהגים
- **ניהול נסיעות**: יצירה, עדכון, מחיקה וצפייה בנסיעות
- **ניהול זמינות נהגים**: נהגים יכולים לעדכן את זמינותם ומיקומם הנוכחי
- **מערכת בקשות נסיעה**: חברות יכולות לשלוח בקשות לנהגים ונהגים יכולים לקבל או לדחות אותן
- **מערכת התראות**: התראות בזמן אמת על עדכוני נסיעות, בקשות חדשות ועוד
- **דוחות וניתוח נתונים**: סטטיסטיקות וניתוח נתונים לחברות ונהגים
- **מערכת דירוג**: דירוג הדדי בין חברות ונהגים לאחר השלמת נסיעות

## טכנולוגיות
- **צד לקוח**: React.js, React Bootstrap, Context API
- **צד שרת**: Node.js, Express.js
- **מסד נתונים**: MySQL
- **אימות**: JSON Web Tokens (JWT)
- **כלים נוספים**: Axios, Jest (בדיקות)

## התקנה והפעלה

### דרישות מקדימות
- Node.js (גרסה 14 ומעלה)
- MySQL (גרסה 5.7 ומעלה)
- npm או yarn

### התקנת צד שרת
1. שכפל את המאגר:
   ```
   git clone https://github.com/yourusername/transport-company.git
   ```

2. עבור לתיקיית השרת:
   ```
   cd transport-company/server
   ```

3. התקן את התלויות:
   ```
   npm install
   ```

4. צור קובץ `.env` והגדר את המשתנים הבאים:
   ```
   DB_HOST=localhost
   DB_USER=your_mysql_username
   DB_PASSWORD=your_mysql_password
   DB_NAME=transport_company
   JWT_SECRET=your_jwt_secret
   PORT=5000
   ```

5. צור את מסד הנתונים:
   ```
   mysql -u your_mysql_username -p < config/database.sql
   ```

6. הפעל את השרת:
   ```
   npm start
   ```

### התקנת צד לקוח
1. עבור לתיקיית הלקוח:
   ```
   cd ../client
   ```

2. התקן את התלויות:
   ```
   npm install
   ```

3. צור קובץ `.env` והגדר את המשתנים הבאים:
   ```
   REACT_APP_API_URL=http://localhost:5000/api
   ```

4. הפעל את האפליקציה:
   ```
   npm start
   ```

## מבנה הפרויקט

### צד שרת
```
server/
├── config/              # קבצי תצורה
│   ├── database.sql     # סכמת מסד נתונים
│   └── db.js            # חיבור למסד נתונים
├── controllers/         # בקרים
├── middleware/          # מידלוור
├── models/              # מודלים
├── routes/              # נתיבי API
├── utils/               # פונקציות שירות
├── __tests__/           # בדיקות
└── server.js            # נקודת כניסה
```

### צד לקוח
```
client/
├── public/              # קבצים סטטיים
├── src/
│   ├── components/      # רכיבי React
│   │   ├── admin/       # רכיבי מנהל
│   │   ├── company/     # רכיבי חברה
│   │   ├── driver/      # רכיבי נהג
│   │   └── common/      # רכיבים משותפים
│   ├── context/         # Context API
│   ├── pages/           # דפים
│   ├── services/        # שירותי API
│   ├── utils/           # פונקציות שירות
│   ├── App.js           # רכיב ראשי
│   └── index.js         # נקודת כניסה
└── package.json
```
